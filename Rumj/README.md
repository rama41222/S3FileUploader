# Rumj API 1.0.0v



##### Requirements
You must have 
1. docker*  
2. Java >= 1.8

*optional 
##### To get started using Docker
```
docker pull rama41222/rumj
docker run -d --net=host --name=rumj rama41222/rumj
```

For mac users 
```
docker pull rama41222/rumj
docker run -d -p 8080:8080 --name=rumj rama41222/rumj
```

##### Run without docker
```
java -jar com-rama41222-Rumj-aws-spring-boot.jar &
```
